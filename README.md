<h3><b>Sobre Mim</b></h3>

💫 Desenvolvedor com foco em soluções escaláveis e raciocínio lógico apurado, unindo software, dados e hardware.

🎯 **Status Atual:** Estudante de Desenvolvimento de Software Multiplataforma (4º semestre) na Fatec Zona Sul e atuando como voluntário na ONG Practical Science Education (PSE) nas frentes de tecnologia e dados de marketing.

⚙️ **Base Técnica:** Técnico em Mecatrônica pela Etec Getúlio Vargas, o que me confere uma base sólida em lógica e sistemas complexos.

📊 **Dados & Estratégia:** Utilizo SQL e Power BI na ONG PSE para transformar dados brutos em dashboards que guiam a tomada de decisão estratégica.

🚀 **Projetos de Destaque:**
* **TechSyn:** Plataforma gamificada de eventos (React, Node.js, MySQL). 🏆 *Vencedor de "Melhor Projeto Web" - Demo Day Instituto PROA 2025.*
* **Reestoque:** E-commerce funcional com busca e checkout integrado à Azure.
* **Aquário Automatizado:** Projeto de IoT unindo hardware e software com Arduino e sensores eletrônicos (C/C++).
* **Aura Mind:** Ecossistema de automação residencial focado na redução da carga cognitiva dos usuários.

🎓 **Certificações de Peso:**
* Qualificação Profissional Desenvolvedor Front-End (640h) – Fatec.
* Programação e Desenvolvimento Web .NET (440h) – Instituto PROA.
* Desenvolvedor Web .NET (320h) – Senac Lapa Tito.
* Micro Certificação em Design Patterns Básico – Fatec/CPS.
